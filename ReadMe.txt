User Operate Guide
*****************************************
-------------
Shortcut KEYS:
-------------
CTRL+S	=	Clock Setting Open.
CTRL+E	=	Exit 7-Clock.
CTRL+T	=	Top Most for other windows.

CTRL(HOLD) + Mouse(Button Pressed) Move = Move Clock Anywhere.
